# SCADYS.io

SCADYS.io develops embedded marine systems with a strong focus on product firmware, reusable embedded components, and developer-facing technical documentation.

## Organization Focus

- Product firmware development using the ESP-IDF framework.
- Reusable embedded software components that can be shared across multiple products.
- Public technical documentation and interface specifications where appropriate.
- Clear separation between open firmware assets and proprietary hardware/design assets.

## Engineering Direction

The SCADYS.io codebase is being consolidated around:

- reusable ESP-IDF components,
- product-specific firmware applications,
- public developer and product documentation, and
- governance assets that apply across the organization.

Legacy firmware from the SixSense Marine and GM Consult repositories is being migrated and refactored into SCADYS.io repositories. Over time, those legacy organizations and repositories are intended to be archived after the migration is complete.

## Open And Proprietary Boundaries

SCADYS.io may open-source firmware and selected technical documentation where this supports the developer community and product ecosystem.

The following categories are intended to remain proprietary:

- hardware design source files,
- manufacturing and supplier information,
- mechanical CAD,
- design-source artwork and UI source assets, and
- other implementation details not required to build, use, or contribute to firmware.

## Governance

SCADYS.io is the current brand. GM Consult Pty Ltd is the operating company. GM Consolidated Holdings Pty Ltd is the holding company, with the strategic intent of quarantining IP ownership at the holding-company level.

## Repositories

This organization includes repositories for:

- shared embedded components,
- product firmware,
- public documentation,
- governance artifacts, and
- supporting engineering tooling.
